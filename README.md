This is the evaluation repository for the paper "Benchmarking Failure - The State (and Decay) of Reentrancy Detection Tools" submitted at S&P 2026 on November 13th, 2025.
The `dataset` folder contains the handcrafted dataset in multiple versions.
The `test_environment` folder contains Hardhat scripts and attacker contracts to verify the dataset.
The `resuts` folder contains results in CSV format both for the tools included in the Smartbugs suite and those not included.
